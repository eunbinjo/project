Operator (연산자)

어떤 연산을 처리하는데 사용하도록 미리 정의된 기호


산술 연산자

산술 연산자는 덧셈, 뺄셈, 곱셈, 나눗셈과 같은 수학 연산을 수행하는 연산자를 말한다.

산술 연산자
구분 연산자 예 설명
이항 산술 + x + y x에 y를 더한다.
- x - y x에서 y를 뺀다.
* x * y x에 y를 곱한다.
/ x / y x를 y로 나눈다.
% x % y x를 y로 나누어 나머지를 구한다.
** x ** y x의 y거듭제곱을 구한다.
단항 산술 ++ x++
++x 		x를 1 증가시킨다.
-- x--
--x 		x를 1 감소시킨다.
단항 부정 - -x x의 부호를 부정한다. (음수->양수 / 양수->음수)

이항 산술 연산자

이항 산술 연산자는 연산을 수행하는데 피연산자가 2개 필요한 연산자이다.

이항 산술 연산자 예시

let sum = 10 + 20; // 30
let sub = 20 - 10; // 10
let multi = 10 * 20; // 200
let div = 10 / 2; // 5
let remain = 10 % 3; // 1
let expon = 2 ** 3; // 8

단항 산술 연산자

단항 산술 연산자는 연산을 수행하는데 피연산자가 1개만 필요한 연산자이다. ++연산자와 --연산자를 합쳐 '증감 연산자'라고도 한다.

단항 산술 연산자 예시

let increment = 10;
increment++;
let decrement = 10;
decrement--;

console.log(increment); // 11
console.log(decrement); // 9

단항 산술 연산자는 변수나 상수에 할당된 데이터로만 연산할 수 있고, 숫자에 바로 사용하는 건 안 된다.

전치 연산과 후치 연산

단항 산술 연산자를 앞에 사용했는지, 뒤에 사용했는지에 따라 결과가 달라진다.

전치 연산

let num = 10;
let subNum = ++num; //전치 연산

console.log(subNum); // 11

변수에 값을 할당하기 전에 1을 증가시킨다.

후치 연산

let num = 10;
let subNum = num++; //후치 연산

console.log(subNum); // 10

값을 증가시키기 전에 먼저 변수에 값을 할당시킨다.

단항 부정 연산자

항상 피연산자 앞에 위치하며, 피연산자의 부호를 부정하는 연산을 수행한다. 쉽게말해, 음수를 양수로 변환하고 양수는 음수로 변환하는 연산을 수행한다. 따라서 숫자형 데이터가 할당된 변수에만 사용할 수 있다.

단항 부정 연산자

let num = -10;
num = -num;

console.log(num); // 10


대입 연산자와 복합 대입 연산자

대입 연산자는 데이터를 대입(할당)하는 연산을 수행하는 연산자를 말하며, 복합 대입 연산자는 산술 연산자와 대입 연산자를 함께 사용해 산술과 할당을 한 번에 수행하는 연산자를 말한다.

대입 연산자와 복합 대입 연산자
구분 연산자 예 설명
대입 연산자 = x = y x에 y를 대입한다.
복합 대입 연산자 += x += y x에 x + y를 대입한다.
-= x -= y x에 x - y를 대입한다.
*= x *= y x에 x * y를 대입한다.
/= x /= y x에 x / y를 대입한다.
%= x %= y x에 x & y를 대입한다.
**= x **= y x에 x ** y를 대입한다.

비교 연산자

비교 연산자는 피연산자를 비교한 뒤, 논리형 값인 참(true), 거짓(false)을 반환하는 연산을 수행한다.

비교 연산자
연산자 예 설명
== x == y x와 y의 값이 같으면 true를 반환한다.
=== x === y x와 y의 값과 타입(자료형)이 같으면 true를 반환한다.
!= x != y x와 y의 값이 다르면 true를 반환한다.
!== x !== y x와 y의 값과 타입(자료형)이 다르면 true를 반환한다.
< x < y x가 y보다 작으면 true를 반환한다.
<= x <= y x가 y보다 작거나 같으면 true를 반환한다.
> x > y x가 y보다 크면 true를 반환한다.
>= x >= y x가 y보다 크거나 같으면 true를 반환한다.

논리 연산자

논리 연산자는 피연산자를 논리적으로 평가한 뒤, 조건에 맞는 피연산자를 반환하는 연산을 수행한다.

논리 연산자
연산자 예 설명
&& (and 연산자) x && y x가 참이면 y를 반환하고, 거짓이면 x를 반환한다.
|| (or 연산자) x || y x가 참이면 x를 반환하고, 거짓이면 y를 반환한다.
! (not 연산자) !x x가 참이면 false를 반환하고, 거짓이면 true를 반환한다.

논리 연산자의 특징

어떤 피연산자든 모두 논리형으로 평가한다. 그러니까 숫자형을 쓰든 문자열을 쓰든 전부 논리 값으로 평가한다.

자바스크립트의 자료형 중 ""(빈 문자열), undefined, null, 0만 거짓으로 평가되고 나머지는 참으로 평가한다.


삼항 연산자

삼항 연산자는 세 항 중 가장 왼쪽에 있는 피연산자의 참, 거짓에 따라 나머지 두 항에 있는 피연산자를 선택적으로 반환하는 연산을 수행한다.

삼항 연산자
연산자 예 설명
?: x ? y : z x가 참이면 y를 반환하고, x가 거짓이면 z를 반환한다.

연산자 우선순위

연산자를 여러 개 사용했을 때, 어떤 연산자를 먼저 연산할지를 결정하는 기준이 된다. 우선순위가 같은 연산자들을 사용했을 경우에는 결합 순서에 따라 연산이 수행된다.

연산자 우선순위
우선순위 연산자 종류 기호 결합 순서
1 그룹 연산자 () 좌 -> 우
대괄호 연산자 [] 좌 -> 우
마침표 연산자 . 좌 -> 우
2 증가 연산자 ++ 우 -> 좌
감소 연산자 -- 우 -> 좌
단항 부정 연산자 - 우 -> 좌
NOT 연산자 ! 우 -> 좌
delete 연산자 delete 우 -> 좌
new 연산자 new 우 -> 좌
typeof 연산자 typeof 우 -> 좌
3 나눗셈 연산자 / 좌 -> 우
곱셈 연산자 * 좌 -> 우
나머지 연산자 % 좌 -> 우
4 덧셈 연산자 + 좌 -> 우
뺄셈 연산자 - 좌 -> 우
5 비교 연산자 <=, <, >, >= 좌 -> 우
6 동등, 일치, 부등, 불일치 ==, ===, !=, !== 좌 -> 우
7 AND 연산자 && 좌 -> 우
8 OR 연산자 || 좌 -> 우
9 삼항 연산자 ? : 좌 -> 우
10 대입(할당) 연산자 = 우 -> 좌
복합 대입 연산자 +=, -=, *=, /=, %=, **= 우 -> 좌
11 멀티 연산자 , 좌 -> 우

연산자 우선순위는 종류가 많고, 결합 순서도 조금씩 다르기 때문에 가능한 한 우선 순위가 가장 높은 그룹의 연산자를 사용해 식의 우선순위를 단순하게 정리하는 것이 좋다.


형 변환

형 변환은 데이터의 자료형이 다른 자료형으로 바뀌는 것을 말한다. 크게 암시적 형 변환과 명시적 형 변환이 있다.

암시적 형 변환

암시적 형 변환 예시

const result = 10 + '10';
console.log(result); // 1010

자바스크립트 내부적으로 숫자형 데이터를 문자형 데이터로 형 변환한 것.

명시적 형 변환

드러나게 형 변환을 하는 것.

명시적 형 변환 예시

let num = 10;
let strNum = '10';
if(String(num) == strNum) {
console.log(`equals`);
}
