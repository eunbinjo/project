즉시 실행 함수(IIFE, Immediately Invoked Function Expression)

함수를 정의하면서 동시에 실행까지 하는 함수.

형식

(function(){})();

한 번만 사용할 함수나 변수를 const 키워드를 사용해 정의해, 어떤 식별자를 더이상 사용할 수 없게 되는 현상을 가리켜 글로벌 스코프가 오염됐다고 표현한다. 이런 경우, 즉시 실행 함수를 사용하면 글로벌 스코프가 오염되는 것을 방지할 수 있다.

즉시 실행 함수 예시

(function init(){
console.log("initialixed!");
})();

즉시 실행 함수는 한 번 실행되고 나면 메모리에 데이터가 남지 않는다. 그래서 위의 init 식별자는 한 번도 사용되지 않은 것처럼 인식된다.

파라미터가 있는 함수도 즉시 실행 함수로 정의해서 실행할 수 있다.

파라미터 함수의 즉시 실행 예시

(function sum(a, b){
console.log(a + b);
})(10, 20);

실행 결과 : 30
