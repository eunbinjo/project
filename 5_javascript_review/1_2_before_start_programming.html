자바스크립트 오류 확인 방법

프로그래밍 언어를 실행하는 방법

Compile(컴파일) 방식
코드를 실행하기 전에 모든 코드를 컴퓨터가 이해할 수 있는 기계어로 한 번 변환한 후 실행하는 방식

Interpreter(인터프리터) 방식
코드를 한 번에 한 줄씩 실행하는 방식

그럼 자바스크립트는?

자바스크립트는 인터프리터 방식으로 실행되기 때문에, 오류가 발생하면 오류가 발생한 시점과 줄 번호를 알려주고, 오류 발생 시점부터 실행을 멈춘다.

오류 메시지와 오류가 발생한 줄 번호를 웹 브라우저의 콘솔창에 표시한다.
