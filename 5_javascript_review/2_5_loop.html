Loop (반복문)

반복문은 지정한 조건이 참(true)로 평가되는 동안 지정한 블록문을 반복해서 실행하는 문법이다.


while 문

자바스크립트 반복문 중 기본. while 문은 특정 조건을 만족하는 동안 블록문을 실행한다.

1부터 9999까지 출력하는 코드 let num = 1;
while (num <= 9999) {
console.log(num);
num++;
}

무한 반복문

반복문을 작성할 때 주의해야 할 점이다. 무한 반복문이란 반복문의 조건이 계속 참으로 평가되어 반복문이 끝나지 않고 무한히 실행되는 것을 말한다.

끝나지 않는 반복문은 프로그램에 치명적인 성능 저하를 일으킬 수 있으므로, 반복문의 조건이 거짓(false)가 되는 순간이 있는지 신경 써야 한다.


do... while 문

do... while 문은 특정 조건이 참으로 평가되는 동안 do 다음에 오는 블록문을 반복 실행한다.

while 문은 실행 전에 조건식을 평가하지만, do while문은 블록문을 무조건 한 번 수행하고 나서 조건식을 평가한다는 특징이 있다.

조건이 false임에도 한 번은 실행된다.

do{
console.log('무조건 한 번은 실행된다.');
} while(false);


for 문

for 문은 횟수를 지정해 지정한 횟수가 끝날 때까지 블록문을 반복 실행하는 반복문으로, 조건식과 증감식이라는 독특한 실행 구조로 되어있다.

for 문의 형식

for (초깃값; 조건식; 증감식){
//블록문
}

위 형식일때, for 문은 초깃값 -> 조건식 -> 블록문(조건식이 참일 경우) -> 증감식 -> 조건식... 순서로 진행된다.

예시
코드 실행 결과
for(let i = 0; i < 5; i++){
console.log(i);
} 		0
1
2
3
4

위 for 문의 실행 순서

1. 반복문이 실행될 때 변수 i의 값을 0으로 초기화 한다.
2. 변수 i의 값이 5보다 작은지 평가한다.
2-1. 참이면 3번으로 간다.
2-2. 거짓이며 반복문을 종료한다.
3. 블록문을 실행한다.
4. 블록문을 실행한 후 변수 i의 값을 1 증가시킨다.
5. 변수 i가 5보다 작은지 다시 평가한다.
6. 평가 결과가 거짓일 때까지 2~4번을 반복한다.

중첩하기

for 문도 중첩해서 사용할 수 있다.

중첩 반복문의 기본이 되는 반복문은 가장 외부에서 실행되는 for문이다. 내부에 중첩된 for문은 외부 for 문의 실행이 종료되면 같이 종료된다.

중첩 예시
코드 실행 결과
for(let i = 0; i < 2; i++){
console.log(`i: ${i}`);
for(let k = 0; k < 2; k++){
console.log(`k: ${k}`);
}
} 		i: 0
k: 0
k: 1
i: 1
k: 0
k: 1

- 가장 외부에 있는 for 문의 초깃값은 반복문이 실행될 때 최초 1번만 설정되지만, 내부 for문의 초깃값은 외부 블록문이 실행될 때마다 새로 설정된다.

위 중첩 반복문의 실행 순서

1. 외부 for 문의 초깃값을 설정한다.
2. 외부 for 문의 조건식을 평가한다.
2-1. 참이면 3번으로 간다.
2-2. 거짓이면 for문을 종료한다.
3. 외부 블록문을 실행한다.
4. 내부 for 문의 초깃값을 설정한다.
5. 내부 for 문의 조건식을 평가한다.
5-1. 참이면 6번으로 간다.
5-2. 거짓이면 내부 for문을 종료한다.
6. 내부 블록문을 실행한다.
7. 내부 for문의 증감식을 실행한다.
8. 내부 for 문의 조건식이 참일 동안 5-7번을 반복한다.
9. 내부 for 문이 종료되면 외부 for문의 증감식을 실행한다.
10. 외부 for 문의 조건식이 참일 동안 2-9번을 반복한다.


for 문과 배열

for 문은 횟수를 지정하는 반복문이라는 특징 때문에, 배열과 같은 자료형을 반복 횟수 용도로 사용할 수 있다.

배열요소에 접근해 값을 출력
코드 실행 결과

let arr = ["banana", "apple", "orange"];

for(let i = 0; i < arr.length; i++){
console.log(arr[i]);
}

banana
apple
orange

배열에 length 속성을 사용하면 배열의 데이터 개수(배열의 길이)를 확인할 수 있는 것을 활용해 만든 코드이다.


for... in

객체 리터럴이나 배열에 반복 접근할 수 있는 반복문.

for... in의 형식

for(가변수 in 배열/객체리터럴){
//블록문
}

for문의 소괄호 안에 in 키워드를 두고, 키워드의 오른쪽에는 탐색의 대상이 되는 배열 또는 객체리터럴을, 왼쪽에는 배열과 객체리터럴을 탐색해 키를 저장할 가변수(임시 변수)를 놓는다.

객체 리터럴을 반복할 경우

가변수에 key 값을 할당할 수 있다.

객체 리터럴의 for...in
코드 실행 결과

let obj = {name: "지젤", age: "20"};

for(let key in obj){
console.log(`${key} : ${obj[key]}`);
}

name : 지젤
age : 20

배열을 반복할 경우

가변수에 index 값을 할당할 수 있다.

배열의 for...in
코드 실행 결과

let arr = ["orange", "apple", "banana"];

for(let index in arr){
console.log(`${index} : ${arr[index]}`);
}

0 : orange
1 : apple
2 : banana

인덱스 순서대로 배열에 반복 접근한 것처럼 보이나, 문법적으로는 for... in문 반복시 배열의 순서대로 접근하는 것을 보장하지 않으므로 주의해야 한다.


break 문

지금까지의 반복문은 '특정 조건이 참으로 평가되는 동안', 또는 '일정 횟수 동안' 반복해서 코드를 실행한다. 반복문이 종료되려면 조건이 거짓으로 평가되거나, 일정 횟수가 끝나야 실행이 종료된다.

그러나 이런 종료 조건을 만족하지 않아도 인위적으로 반복문을 종료하는 방법이 있으니, 그것이 break 문이다.

break 문의 사용
구분 코드 실행 결과
일반 for 문

for(let i = 0; i < 10; i++){
console.log(i);
if(i === 5) break;
}

0
1
2
3
4
5

for... in 문

let obj = {name:"지젤", age:"20"};

for(let key in obj){
if(key === age) break;
console.log(obj[key]);
}

지젤

이렇게 break문으로 반복문을 즉시 종료할 수 있기에, 반복문 내부에 break문을 바로 사용하기 보다는 if 문으로 처리하는 것이 일반적이다.


continue 문

break 문은 반복문을 즉시 종료하지만, continue 문은 해당하는 반복문을 건너뛰고 실행하라는 명령이다.

contonue 문의 사용
코드 실행 결과

for (let i = 0; i <= 10; i++){
if(i % 2 === 1) continue;
console.log(i);
}

2
4
6
8
10
