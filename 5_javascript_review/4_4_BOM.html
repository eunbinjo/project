브라우저 객체 모델 (BOM, Browser Object Model)

표준 내장 객체는 자바스크립트에 내장된 객체이다. 그런데 자바스크립트 언어 사양에 포함되지 않고, 웹 브라우저에서 제공하는 객체도 존재한다. 이것을 브라우저 객체 모델이라고 한다.

브라우저 객체 모델
window
document location history navigator screen
form cookie links/anchors images

window : 웹 브라우저가 열릴 때마다 생성되는 최상위 관리 객체
document : 웹 브라우저에 표시되는 HTML 문서 정보가 포함된 객체
location : 웹 브라우저에 현재 표시된 페이지에 대한 URL 정보가 포함된 객체
history : 웹 브라우저에 저장된 방문 기록이 포함된 객체
navigator : 웹 브라우저 정보가 포함된 객체
screen : 웹 브라우저의 화면 정보가 포함된 객체


window 객체의 속성과 메서드

window 객체에는 웹 브라우저의 기능과 요소들을 제어할 수 있는 여러 속성과 메서드가 존재한다. 주요 속성과 메서드를 알아보자.

window 객체의 주요 속성과 메서드
구분 설명
속성 innerWidth 웹 브라우저 화면의 너비를 px(픽셀) 단위로 나타낸다.
innerHeight 웹 브라우저 화면의 높이를 px 단위로 나타낸다.
outerWidth 웹 브라우저 창의 너비를 px 단위로 나타낸다.
outerHeight 웹 브라우저 창의 높이를 px 단위로 나타낸다.
screenTop / screenY 웹 브라우저 위쪽 면과 모니터의 간격을 px 단위로 나타낸다.
screenLeft / screenX 웹 브라우저 왼쪽 면과 모니터의 간격을 px 단위로 나타낸다.
pageXOffset / scrollX 웹 브라우저의 수평 스크롤 위치를 px 단위로 나타낸다.
pageYOffset / scrollY 웹 브라우저의 수직 스크롤 위치를 px 단위로 나타낸다.
메서드 alert() 알림창을 표시한다.
confirm() 확인창을 표시한다.
prompt() 입력창을 표시한다.
open() 새로운 웹 브라우저 창을 연다.
close() 웹 브라우저 창을 닫는다.
setTimeout() 일정 시간(ms) 뒤에 콜백 함수를 한 번만 실행한다.
setInterval() 일정 시간(ms)마다 콜백 함수를 반복적으로 실행한다.
clearInterval setInterval() 메서드로 반복 실행되는 함수를 중지한다.
scrollTo() 웹 브라우저의 스크롤을 특정 위치만큼 이동한다.
scrollBy() 웹 브라우저의 스크롤을 현재 위치에서 상대적 위치로 이동한다.

window 객체의 기본 속성

window 객체의 속성은 주로 웹 브라우저의 창과 관련 있는 것이 많다.

console 창을 열고 위 버튼을 누르면 기본 속성 확인 가능.


웹 브라우저에서 새 창 제어하기

window 객체의 open() 메서드는 웹 브라우저에서 새로운 창을 여는 데 사용한다.

open()

형식
window.open(경로, 이름, 속성);




코드 설명

function popup(){
window.open('4_5_popup.html', '팝업', 'width=200, height=100');
}

첫 번째 파라미터 : 4_5_popup.html

새 창에서 열릴 파일의 경로를 나타낸다.

두 번째 파라미터 : '팝업'

새 창의 이름을 '팝업'이라고 지정하겠다는 뜻. 웹 브라우저 탭에 보이는 것은 아니고, 내부적으로 구분하는 용도다. -> open()메서드는 같은 이름의 팝업 창을 기본적으로 1개만 연다.

따라서 이 파라미터의 이름을 빈 문자열('')로 지정하면, 같은 팝업창이 여러 개 열리게 할 수도 있다.

세 번째 파라미터 : 'width=200, height=100'

새 창의 속성을 지정하는 값이다. 세 번째 파라미터에 사용할 수 있는 속성은 다음과 같다.

속성 예시 설명
width width=400 웹 브라우저의 너비를 px 단위로 지정
height height=400 웹 브라우저의 높이를 px 단위로 지정
left left=400 웹 브라우저 왼쪽에서의 위치를 px 단위로 지정
top top=400 웹 브라우저 위쪽에서의 위치를 px 단위로 지정

close()

open() 메서드로 연 새 창은 window 객체의 close() 메서드로 닫을 수 있다.

형식
window.close()


웹 브라우저의 스크롤 이동하기

스크롤 이동
메서드 형식 설명
scrollTo() window.scrollTo(x좌표, y좌표) 웹 브라우저의 스크롤 위치를 특정 좌표로 이동하게 한다.
scrollBy() window.scrollBy(x좌표, y좌표) 웹 브라우저의 스크롤을 현재 위치에서 상대적인 위치로 이동하게 한다.


scrollTo는 여러 번 호출해도 작성한 좌표로만 스크롤이 된다.

형식 : window.scrollTo(100, 400)
window.scrollTo({left:100, top:400})

scrollBy를 사용하면 호출 할 때마다 작성한 좌표만큼씩 스크롤이 된다.

형식 : window.scrollBy(100, 400)
window.scrollBy({left:100, top:400})

behavior 속성

위의 두 메서드는 좌표를 객체 리터럴로 전달할 수도 있는데, 이때 behavior 속성도 함께 전달할 수 있다.

형식 : window.scrollBy({top:400, behavior:'smooth'})
