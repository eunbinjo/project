노드 추가/삭제하기

DOM 트리에 새로운 노드를 생성하고, 생성한 노드를 기본 DOM 트리와 연결하면 동적으로 새로운 요소를 화면에 추가할 수 있다.


노드 추가하기

새로운 요소를 화면에 추가하려면 먼저 DOM 트리에 새로운 노드를 생성하는 작업을 해야한다. 그 이후에 생성한 노드를 기존의 DOM 트리에 연결한다.

노드 추가 시 사용하는 메서드
구분 메서드 설명
노드 생성 createElement() 요소 노드를 생성한다.
createTextNode() 텍스트 노드를 생성한다.
createAttribute() 속성 노드를 생성한다.
노드 연결 [기준 노드].appendChild([자식 노드]) 기준 노드에 자식 노드를 연결한다.
[기준 노드].setAttributenode([속성 노드]) 기준 노드에 속성 노드를 연결한다.

노드 생성 및 연결 예제 페이지

위 페이지에 적용한 스크립트

//요소 노드 생성하기
const aEl = document.createElement("a");

//기존의 돔 트리와 연결
//body 요소 노드의 자식 노드로 연결
document.body.appendChild(aEl);

//단순히 요소만 연결했으므로, 웹 브라우저에 보이는 텍스트 노드를 연결해본다.
//텍스트 노드 추가하기
//a 요소 노드의 자식으로 연결
const txtEl = document.createTextNode("귀염둥이펭귄 지젤");
document.querySelector("a").appendChild(txtEl);

//a 태그의 속성 노드 연결 (href)
//속성노드는 a태그의 자식으로 연결하는 것이 아니다! (다른 메서드 사용)
const hrefAttr = document.createAttribute("href");
hrefAttr.value = "giselle.png";
document.querySelector("a").setAttributeNode(hrefAttr);

//a 태그의 속성 노드 연결(target)
const target = document.createAttribute("target")
target.value = "_blank";
document.querySelector("a").setAttributeNode(target);


//번외
//img 노드 추가하기
//1. 요소노드 생성
const img = document.createElement("img");
//2. 기존 돔 트리와 연결
document.body.appendChild(img)
//3. img 요소노드의 속성 노드인 src를 연결
const imgEl = document.createAttribute("src");
imgEl.value = "giselle.png";
document.querySelector("img").setAttributeNode(imgEl);


노드 삭제하기

노드를 추가할 수 있다면 당연히 삭제하는 것도 가능하다. 이때는 removechild() 메서드를 사용한다.

removeChild() 메서드는 부모 노드에 연결된 자식 노드를 삭제한다.

형식 [부모 노드].removeChild([자식 노드]);
